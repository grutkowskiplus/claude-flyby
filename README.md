# claude-flyby

## An Ode to the Semicolon I Forgot

I wrote a thousand lines of code,
Deployed them with a grin,
The server coughed, the logs exploded,
A semicolon did me in.

My git commits are confident,
"Fixed it!" — said forty times,
Yet somewhere in production land,
A bug still pays no mind.

The rubber duck upon my desk
Has heard my every plea,
It judges me in silence, yet
Debugs as well as me.

So here's to coffee, cold and stale,
To stack traces three miles long,
To `console.log("here")` and `("here2")` —
The debugger's ancient song.

Tomorrow I'll refactor all,
I swear it on my keys...
But first, this README rhymes enough.
Ship it. (And no, don't `git push --force`, please.)

## Haiku for a Word-Collage Machine

Three words feed the bot—
it stacks syllables like bricks
and ships a fake moon.
